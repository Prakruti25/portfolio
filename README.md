# Prakruti Patel — Personal Portfolio

An elegant, minimal portfolio site for **Prakruti Patel**, Computer Science senior at Indiana University Bloomington.

🔗 **Live site:** _coming soon_

## About

A single-page portfolio built from scratch with HTML, CSS, and vanilla JavaScript. No frameworks, no build tools — just clean code and thoughtful design.

## Design

- **Palette:** Pale sage and blush accents on warm cream — an airy, feminine aesthetic
- **Typography:** Cormorant Garamond (display serif) paired with Jost (clean geometric sans)
- **Layout:** Generous whitespace, editorial composition, arched photo frame
- **Interactions:** Subtle scroll reveals, gentle hover states, no unnecessary motion

## Sections

- Hero with arched portrait
- About with pull-quote
- Skills organized by category
- Work experience timeline
- Selected projects with soft gradient thumbnails
- Contact

## Tech

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (IntersectionObserver for reveals)
- Google Fonts
- No dependencies, no build step

## Run locally

Just open `index.html` in any modern browser.

```bash
open index.html
```
